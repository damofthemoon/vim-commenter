# vim-commenter

# Introduction

Vim-Commenter is a Neovim plugin to comment a line or a portion of lines
easily.

# Installation

Use [Vim-Plug](https://github.com/junegunn/vim-plug) or any other plugin
manager to install it.

```vim
Plug 'damofthemoon/vim-commenter'
```
